# Automated Product Categorization and Sales Forecasting

Published ML research — CSA 2024 International Conference on Computer Science and Applications.

## Stack
Python · LSTM · TF-IDF · NLP · Deep Learning · Scikit-learn

## What it does
- Classifies products into categories from raw text descriptions using a hybrid LSTM + TF-IDF model
- Forecasts sales trends using deep learning trained on large-scale real-world product datasets
- Improved categorization accuracy by 18% over baseline through systematic architecture experimentation

## Publication
Harshini Gande. *"Automated Product Categorization and Sales Forecasting using Machine Learning and Deep Learning."*
CSA 2024 International Conference on Computer Science and Applications.

## Approach
- Compared classical ML (TF-IDF + logistic regression) against deep learning (LSTM) architectures
- Used holdout validation and cross-validation across experimental iterations
- Documented all methodology to peer-review-quality standards
